# Light-app

## Stack
- React
- Next
- Typescript
- Drizzle
- TailwindCSS
- ESLint (Next.js)
- Prettier
- pnpm
- @next/env (custom dotenv)

## Running tools
```bash
# Running lint
pnpm lint

# Running prettier (you should set up prettier on save in your editor)
pnpm exec prettier . --write
# do not use `pnpm prettier --write .` because it will use the global prettier
```
